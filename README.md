# RUB Markdown Slides

Create beautiful, customized Reveal.js presentations using simple Markdown and YAML Front Matter.

## Remote Usage via HedgeDoc
You can host your Markdown in a HedgeDoc pad (we suggest `md.fachschaften.org`) and load it dynamically without needing any local setup!
Open `index.html` in your browser for a convenient landing page that fetches your live pad automatically and processes it into slides.

## Offline Usage
You can easily use this tool completely offline! Simply modify the included `slides.md` file locally to build your presentation. 

To view your slides, start any basic web server in this directory, for instance using Python:

```bash
python3 -m http.server
```

Then open [http://localhost:8000/slides.html](http://localhost:8000/slides.html) in your browser.
