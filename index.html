RUB Markdown Slides Generator

Create beautiful, university-branded Reveal.js presentations directly from Markdown!

Load from HedgeDoc Pad

You can use a HedgeDoc pad (we suggest md.fachschaften.org) to write and collaborate on your slides online.

Sample slides: https://md.fachschaften.org/WM4DIek2QVKHnfx5mRwPFA

Offline Usage & Source Code

Offline usage is easily possible! Simply clone the directory, modify the slides.md file, and serve via any basic webserver. For example, using Python:

python3 -m http.server

Source Code is available:

https://git.noc.ruhr-uni-bochum.de/slides/rub-markdown-slides

Note: This project is community-made and is not an official service of the Ruhr-Universität Bochum.

How to Save as PDF

Your presentation will open in Print Mode.

Wait for it to load, then press CTRL + P (or CMD + P) to open the browser print dialog.

Select "Save as PDF" globally.

(Be sure to set margins to 'None' and enable background graphics for best results!)
